# 💼 Krishan Maduwantha — Portfolio

A modern, responsive personal portfolio website built with **React** and **Vite**. Showcases my projects, skills, and professional journey with smooth animations and a clean design.

## ✨ Features

- Fully responsive — works on all screen sizes
- Smooth scroll navigation with active link highlighting
- Animated hero section with floating cards
- Projects section with live demo and source code links
- Skills grid with hover effects
- Contact section with social links
- Clean dark footer

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/beruwalage-krishan-maduwantha/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run locally
```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

### 4. Build for production
```bash
npm run build
```
